# [ 비전기반 스마트 베이커리 운영 플랫폼 ]

 

## 프로젝트 주제

i) 탑뷰 카메라로 트레이 위 상품을 자동 인식해 캐셔리스 결제 시스템을 구축하고, ii) 매장 CCTV 분석으로 혼잡·테이블 회전·안전·방범 등의 운영 지표를 자동 트래킹하는 ‘비전 기반 스마트 카페 운영 플랫폼’ 구현

## 문제 정의 - 베이커리 운영 상 비효율 요소 존재

트레이에 여러 빵/디저트가 올라가며 결제 속도가 매출(피크타임 처리량)에 직접적 영향

상품 외형이 유사(베이글 맛, 빵 종류)하여 숙련된 직원도 제품의 외형을 구분하고 가격을 외우는 일이 어려움

혼잡 상황/대기열 파악, 테이블 회전, 위험 상황(넘어짐/분쟁/도난), 재고 추정 등 매장운영 시 인력 의존도가 높음

## 프로젝트 목표

본 프로젝트는 베이커리 환경을 대상으로,
CCTV 영상과 딥러닝 기반 컴퓨터 비전 기술을 활용하여
① 빵 자동 인식 및 결제 요청,
② 비정상 행동(도난·미결제 반출) 탐지,
③ 매장 운영 효율 향상을 동시에 달성하는 통합 인공지능 시스템을 구현하는 것을 목표로 한다.



### 캐셔리스 결제 비전 시스템 구축

트레이/결제선반 위 상품을 탐지·세분류·개수 산출 → 가격 DB 매칭 → POS/영수증/결제까지 연결

결제 도메인 특성상 오결제율을 구조적으로 0에 수렴시키는 안전장치 포함

### CCTV 분석 기반 매장 효율적 관리

매장 카메라로 인원/대기열/체류시간/테이블 점유/청소 필요/이상상황을 이벤트화하여
관리자 대시보드에서 운영 상 의사결정이 가능한 지표와 알림을 제공

매출 통계와 재고 현황 등 운영 지표 조회 페이지 포함
